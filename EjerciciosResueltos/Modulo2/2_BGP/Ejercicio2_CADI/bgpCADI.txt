Router0
------------------------------------------------------
en
config t

interface G0/0/0
ip address 100.0.0.1 255.255.255.0

interface G0/0/1
ip address 200.1.1.1 255.255.255.0

interface G0/1/0
switchport access vlan 7
switchport mode access
switchport nonegotiate

interface Vlan7
mac-address 0000.0cc5.5902
ip address 200.7.7.1 255.255.255.0

router bgp 65000
neighbor 200.1.1.2 remote-as 65100
neighbor 200.7.7.2 remote-as 65060
network 100.0.0.0 mask 255.255.255.0
# network 200.1.1.0 mask 255.255.255.0 

Al dar de alta esta red, la red amarilla, ya 
conoce la interface más cercana a la red 
100.0.0.0, ya sabe como ir de regreso.
Si desactivamos la interface 200.1.1.1, al intentar
conectarse a la red amarilla 100.3.3.1 se va a ir por la
red inferior.
network 200.7.7.0 mask 255.255.255.0

neighbor establece adjacencia
network que redes quiero anunciar.
Primero anuncio redes y luego recibo tráfico.
No anuncies redes desde las cuales no quieres recibir
tráfico.
Al anunciar la red 200.1.1.0 me estarían penalizando.
