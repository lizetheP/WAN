Tarea 4. Configuración de GRE 
.
.

      Objetivos:  
    • Configurar túneles GRE.
    • Verificar conectividad.

   Modalidad:  
    • Individual.
 
     Instrucciones:
    • Ejecuta la aplicación de Packet Tracer.
    • Abre el archivo 4 GRE.pka
    • Si deseas revisar las instrucciones en español: 4_Configuring_GRE.docx
    • Configura tu User Profile con tu matrícula, nombre y correo, como se muestra en la figura:

                 

    • Lleva a cabo todas las tareas que indica la práctica para que llegues al 100%.
    • Utiliza la línea de comandos de cada equipo de interconexión y realiza la configuración.
    • Para comprobar la configuración exitosa de cada equipo de interconexión de esta práctica, realiza las pruebas de conectividad indicadas en la descripción de la actividad.

                

 
     Recursos:

    • 9_VPNs

       Especificaciones de entrega
 
    • Formato de entrega: Un archivo pka
    • Nombre de los entregables: 4_GRE_matrícula.pka
    • Medio de entrega: Se entrega en la sección de Tarea 4. Configuración de GRE
